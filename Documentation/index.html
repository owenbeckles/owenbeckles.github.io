Samuel - Personal Portfolio Template

Introduction

Samuel is Professional, Clean, Creative eye-catching vCard/resume Template. You can use this template for vCard, Resume, Portfolio, CV, Photography, Freelancer, Art and many more. Its design is up to modern design trends. The template has built with latest Bootstrap v4.5.2, Sass. It’s 100% responsive with clean codes. All components used in the template are well commented, w3 validated and are well documented.

Please feel free to get back to me in case if you are having any question or feedback.
